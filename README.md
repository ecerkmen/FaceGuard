FaceGuard is a real-time facial recognition security system that sends Telegram notifications when any face other than the designated owner is detected at the door. It captures face data for registered individuals, trains a Support Vector Machine (SVM) classifier on 128-dimensional facial encodings generated by dlib, and runs continuous live inference through a webcam feed. Detections are classified either as a known registered individual or as an unknown person, with each triggering a Telegram alert containing both a text message and a photo of the detected face.
